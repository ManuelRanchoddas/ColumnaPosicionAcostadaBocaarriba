# ColumnaPosicionAcostadaBocaarriba
Modelado matematico de la columna vertebral en distintas posiciones, aqui se encontraran en primera instancia el modelado de una columna de una niña de 7 año acostada boca arriba, se plantean las ecuaciones que rigen el sistema atra vez de Open Modelica, se varia la posicion del angulo para encontrar distintos valores de fuerzas en las vertebras L4 y L5 que son las que soportan el peso en un ser humano.
